# iOS Tic Tac Toe Application

## Description
- Created an application based on the original Tic Tac Toe game.

## Application Summary
- Engage in single-player battles against AI with adjustable difficulty levels, from easy to hard.
- Challenge friends in local multiplayer mode for classic head-to-head matches on a single device.
- Take the competition online with the innovative online multiplayer mode, where hosts generate unique codes to create dedicated rooms, enabling seamless connections for guests.
- Additionally, users can customize their experience through the settings page, with options to remove ads, switch between light and dark mode, manage account settings including deletion, and verify email status.

## Topic Covered
- Inner workings of simple gameplay and logic
- Copy to clipboard for text message or other meduims
- User Friendly UI
- Real-time interaction handling
- Completion Handling
- Escaping Closures
- User Accounts
  - Account Creation and Sign Up Process
  - Account Login
  - Account Authentication
  - Account Credential Caching for Quick Login
- NoSQL Databases
  - Asynchronus Database Data Handling
  - Database Data Parsing
  - 
## Technologies Used
- UIKit
- UIStoryboarding
- Cocoa Pods
- Google Firebase Auth (Secure Logins, Account Creation, Email Verification) 
- Google Firebase Storage
- Google Firebase Realtime Database (Database Redundancy Checking for Unique Room IDs and Multiplayer Database Structuring)
